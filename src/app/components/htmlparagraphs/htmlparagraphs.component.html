Parrafos en HTML

El elemento HTML < p > < /p > un párrafo.

Un párrafo siempre comienza en una nueva línea y los navegadores agregan automáticamente un espacio en blanco (un margen) antes y después de un párrafo.




Ejemplo

< p > Este es un parrafo de texto en HTML < /p >

< p > Este es otro parrafo de texto en HTML p>

El resultado es este

Este es un parrafo en HTML

Este es otro parrafo de texto en HTML


Pantalla HTML

Con HTML, no puede cambiar la visualización agregando espacios adicionales o líneas adicionales en su código HTML. El navegador eliminará automáticamente cualquier espacio y línea adicional cuando se muestre la página.

Ejemplo

< p > Este párrafo
contiene muchas lineas
en el código fuente,
pero el navegador
lo ignora < /p >


< p > Este párrafo
contiene
muchos espacios
en el
código fuente,
pero el
navegador
lo
ignora < /p >


Reglas horizontales HTML

La etiqueta < hr > define una ruptura temática en una página HTML y se muestra con mayor frecuencia como una regla horizontal.

El elemento < hr > se usa para separar contenido (o definir un cambio) en una página HTML.

Ejemplo

< h1 > Este es encabezado 1 < /h1 >
< p > Este es un texto. < /p >
< hr >
< h2 > Este es encabezado 2 < /h2 >
< p > Este es otro texto. < /p >
< hr >

El resultado es el siguiente

Este es encabezado 1

Este es un texto

Este es encabezado 2

este es otro texto

La etiqueta < hr > es una etiqueta vacía, lo que significa que no tiene una etiqueta final.


Saltos de línea HTML

El elemento HTML < br > define un salto de línea.

Use < br > si desea un salto de línea (una nueva línea) sin comenzar un nuevo párrafo

Ejemplo

< p > Este es < br > un parrafo< br > con salto de linea < /p >

El resultado es el siguiente

Este es
un parrafo
con salto de linea
