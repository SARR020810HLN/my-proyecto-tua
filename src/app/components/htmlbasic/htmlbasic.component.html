Ejemplos HTML Basic

En este capítulo mostraremos algunos ejemplos básicos de HTML. No se preocupe si usamos etiquetas que aún no conoce.

Documentos HTML

HTML (HyperText Markup Language) es el código que se utiliza para estructurar una página web y su contenido. Por ejemplo, el contenido podría estructurarse dentro de un conjunto de párrafos, una lista de puntos con viñetas o usando imágenes y tablas de datos. Como sugiere el título, este artículo le dará una comprensión básica de HTML y sus funciones.

Todos los documentos HTML deben comenzar con una declaración de tipo de documento: < !DOCTYPE html >.
El documento HTML en sí comienza con< html > y termina con < / html >.
La parte visible del documento HTML está entre < body > y < /body >.


< !DOCTYPE html > — tipo de documento. Es un preámbulo obligatorio. En la noche de los tiempos, cuando HTML era joven (alrededor de 1991/92), los doctypes estaban destinados a actuar como enlaces a un conjunto de reglas que la página HTML tenía que seguir para ser considerada un buen HTML, lo que podría significar la verificación automática de errores y otras cosas útiles. Sin embargo, en estos días, no hacen mucho y básicamente solo se necesitan para asegurarse de que su documento se comporte correctamente. Eso es todo lo que necesitas saber por ahora.
< html >< /html > — el elemento < html >. Este elemento envuelve todo el contenido de la página completa y, a veces, se lo conoce como el elemento raíz.
< head>< /head> — el elemento < head>. Este elemento actúa como un contenedor para todo lo que desea incluir en la página HTML que no es el contenido que muestra a los visitantes de su página. Esto incluye cosas como palabras clave y una descripción de la página que desea que aparezca en los resultados de búsqueda, CSS para diseñar nuestro contenido, declaraciones de conjuntos de caracteres y más.
< meta charset="utf-8">: este elemento establece el conjunto de caracteres que debe usar su documento en UTF-8, que incluye la mayoría de los caracteres de la gran mayoría de los idiomas escritos. Esencialmente, ahora puede manejar cualquier contenido de texto que pueda poner en él. No hay razón para no configurar esto y puede ayudar a evitar algunos problemas más adelante.
< title>< /title> — el elemento < title>. Esto establece el título de su página, que es el título que aparece en la pestaña del navegador en la que se carga la página. También se usa para describir la página cuando la marca como favorita.
< body>< /body> — el elemento < body>. Contiene todo el contenido que desea mostrar a los usuarios web cuando visitan su página, ya sea texto, imágenes, videos, juegos, pistas de audio reproducibles o cualquier otra cosa.

Ejemplo

< !DOCTYPE html>
< html >
< body >
< h1 >My First Heading< /h1 >
< p>My first paragraph.< /p >
< /body>
< /html>

La Declaración < !DOCTYPE >

La declaración < !DOCTYPE > representa el tipo de documento y ayuda a los navegadores a mostrar correctamente las páginas web.
Solo debe aparecer una vez, en la parte superior de la página (antes de cualquier etiqueta HTML).
La declaración < !DOCTYPE > no distingue entre mayúsculas y minúsculas.
La declaración < !DOCTYPE > para HTML5 es:

< !DOCTYPE html >
