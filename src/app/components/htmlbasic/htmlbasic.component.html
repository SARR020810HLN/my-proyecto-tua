Ejemplos HTML Basic

En este capítulo mostraremos algunos ejemplos básicos de HTML. No se preocupe si usamos etiquetas que aún no conoce.

Documentos HTML

Todos los documentos HTML deben comenzar con una declaración de tipo de documento: < !DOCTYPE html >. El documento HTML en sí comienza con < html > y termina con < / html >. La parte visible del documento HTML está entre < body > y < /body >.
