Formato de texto

HTML contiene varios elementos para definir texto con un significado especial.



HTML proporciona etiquetas de formato especiales para formatear texto en la página. Podemos usar estas etiquetas para modificar la forma en que se muestra el texto al usuario. Por ejemplo, podemos poner en negrita algunos elementos de texto, o podemos poner en negrita y subrayar elementos importantes, etc.

Ejemplos

Podemos poner texto en negrita usando la etiqueta < b > texto en negrita < /b >

Podemos poner texto en cursiva usando la etiqueta < i > texto esta en cursiva < /i >

Podemos poner un subindice y un superindice usando la etiqueta < sub > subindice < /sub > y < sup > superindice < /sup >

< b > Este texto esta en negrita < /b >

< i > Este texto esta en cursiva < /i >

< p > Este es un < sub > subindice < /sub > y < sup > superindice < /sup > < /p >

El resultado es el siguiente

Este texto esta en negrita

Este texto esta en cursiva

Este es subindice y superindice


HTML < small > Element

El elemento HTML < small> < /small > define el texto mas pequeño

Ejemplo

< p > Este texto es normal < /p >

< small > Este texto esta enfatizado < /small >

El resultado es el siguiente

Este texto es normal

Este texto esta enfatizado


HTML < del > Element

El elemento HTML < del > define el texto que se ha eliminado de un documento. Los navegadores generalmente marcarán una línea a través del texto eliminado:

Ejemplo

< p > Mi color favorito es < del > azul < /del > rojo < /p >

El resultado es el siguiente

Mi color favorito es azul rojo


HTML < ins > Element

El elemento HTML< ins > define un texto que se ha insertado en un documento. Los navegadores normalmente subrayarán el texto insertado:

Ejemplo

< p > Mi color favorito es < del > azul < /del > < ins > rojo < /ins > < /p >

El resultado es el siguiente

My favorite color is blue red.
