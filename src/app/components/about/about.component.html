Etiquetas html de h1 a h6

¿ Que es la etiqueta h1?
h1 Etiqueta de titulo

La etiqueta h1 también conocida como "etiqueta de título", es la encargada de representar todo el contenido de una página dentro de un sitio web. Se considera la etiqueta más importante ya que debe de incluir la palabra clave del tema principal y no debe de existir más de una etiqueta por página.

¿ Que es la etiqueta h2 ?
h2 Encabezado de subtitulos

La etiqueta h2 también conocida como "Etiqueta de subtítulo", es la encargada de dividir el contenido de la página e indicar los componentes del tema propuesto en la etiqueta h1. Se considera la segunda etiqueta más importante y puede existir más de una etiqueta dentro de una misma página.

¿ Que es la etiqueta h3, h4, h5, h6 ?
h3, h4, h5, h6 Subtema de Subtitulos

Las etiques h3, h4, h5, h6 tatambién conocidas como "Subtemas de Subtítulo", son las responsables de captar las principales ideas que se representan en la etiqueta H2 dentro de una página web. Puede existir más de una de estas etiquetas dentro de la página y generalmente se utilizan para señalar listas o contenido dentro del footer y el menú. Ciertamente, no son tan considerados como parámetros dentro del posicionamiento. Sin embargo, suelen ser bastante útiles para reforzar ideas y generar ejemplos.
