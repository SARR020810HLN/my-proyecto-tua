HTML Elements

Un elemento HTML se define mediante una etiqueta de inicio, algo de contenido y una etiqueta de finalización.




HTML Elements

El elemento HTML es todo, desde la etiqueta de inicio hasta la etiqueta final: <

< tagname > El contenido va aquí...< /tagname >

Ejemplos

< h1 > Mi primer encabezado < /h1 >

< p > Mi primer parrafo < /p >

El resultado es el siguiente

Mi primer encabezado

mi primer parrafo


Nested HTML Elements

Los elementos HTML se pueden anidar (esto significa que los elementos pueden contener otros elementos).

Todos los documentos HTML constan de elementos HTML anidados.

El siguiente ejemplo contiene cuatro elementos HTML ( html, body, h1 y p ).

Ejemplo

< !DOCTYPE html >
< html >
< body >
< h1 > Mi primer encabezado < /h1 >
< p > Mi primer parrafo < /p >
< /body >
< /html >


El resultado es el siguiente

Mi primer encabezado

Mi primer parrafo


Empty HTML Elements

Los elementos HTML sin contenido se denominan elementos vacíos.

La etiqueta< br > define un salto de línea y es un elemento vacío sin una etiqueta de cierre:

Ejemplos

< p > Este es un < br > parrafo con salto de linea.< /p >

El resultado es el siguiente

Este es un
parrafo con salto de linea
