HTML Quotation and Citation Elements

Las citas han sido algo muy bien asentado en HTML desde sus comienzos, puede que debido a su nacimiento en un entorno académico, probablemente el ámbito del conocimiento humano dónde más se utilizan las citas y referencias.




HTML < blockquote > for Quotations

El elemento HTML < blockquote > define una sección que se cita de otra fuente.

Los navegadores suelen aplicar sangría a los elementos < blockquote >

Ejemplo

< p > Los navegadores suelen aplicar sangría a los elementos de comillas en bloque. < /p >

< blockquote cite="http://www.worldwildlife.org/who/index.html" >
Durante casi 60 años, WWF ha estado protegiendo el futuro de la naturaleza. WWF, la organización de conservación líder en el mundo, trabaja en 100 países y cuenta con el apoyo de más de un millón de miembros en los Estados Unidos y cerca de cinco millones en todo el mundo.
< /blockquote >

El resultado es el siguiente

Los navegadores suelen aplicar sangría a los elementos de comillas en bloque.

Durante casi 60 años, WWF ha estado protegiendo el futuro de la naturaleza. WWF, la organización de conservación líder en el mundo, trabaja en 100 países y cuenta con el apoyo de más de un millón de miembros en los Estados Unidos y cerca de cinco millones en todo el mundo.

HTML < q > for Short Quotations

La etiqueta HTML < q > define una cita corta.

Browsers normally insert quotation marks around the quotation.

Ejemplo

< p > Los navegadores suelen insertar comillas alrededor del elemento q. < /p >

< p > El objetivo de WWF es: < q > Construir un futuro donde las personas vivan en armonía con la naturaleza.< /q > < /p >

El resultado es el siguiente

Los navegadores suelen insertar comillas alrededor del elemento q.

El objetivo de WWF es: Construir un futuro donde las personas vivan en armonía con la naturaleza.


HTML < abbr > for Abbreviations

La etiqueta HTML < abbr > define una abreviatura o un acrónimo, como "HTML", "CSS", "Sr.", "Dr.", "ASAP", "ATM".

Marcar abreviaturas puede brindar información útil a los navegadores, sistemas de traducción y motores de búsqueda.

Sugerencia: use el atributo de título global para mostrar la descripción de la abreviatura/acrónimo cuando pase el mouse sobre el elemento.

Ejemplo

< p > La< abbr title="World Health Organization" > WHO < /abbr > was founded in 1948. < /p >

< p > Marking up abbreviations can give useful information to browsers, translation systems and search-engines. < /p >

El resultado es el siguiente

The WHO was founded in 1948.

Marking up abbreviations can give useful information to browsers, translation systems and search-engines.


HTML < address > for Contact Information

La etiqueta HTML < dirección > define la información de contacto del autor/propietario de un documento o artículo.

La información de contacto puede ser una dirección de correo electrónico, URL, dirección física, número de teléfono, identificador de redes sociales, etc.

El texto en el elemento < dirección > generalmente se presenta en cursiva, y los navegadores siempre agregarán un salto de línea antes y después del elemento < dirección >

Ejemplo

< p > El elemento de dirección HTML define la información de contacto (autor/propietario) de un documento o artículo.< /p >

< address > Written by John Doe.< br > Visit us at:< br > Example.com < br > Box 564, Disneyland < br > USA < /address >

El resultado es el siguiente

El elemento de dirección HTML define la información de contacto (autor/propietario) de un documento o artículo.

Written by John Doe.
Visit us at:
Example.com
Box 564, Disneyland
USA

HTML < bdo > for Bi-Directional Override

BDO significa anulación bidireccional.

La etiqueta HTML < bdo > se utiliza para anular la dirección del texto actual:

Ejemplo

< p > Si su navegador admite anulación bidireccional (bdo), la siguiente línea se escribirá de derecha a izquierda (rtl): < /p >

< bdo dir="rtl" > Esta línea se escribirá de derecha a izquierda < /bdo >

El resultado es el siguiente

Si su navegador admite anulación bidireccional (bdo), la siguiente línea se escribirá de derecha a izquierda (rtl):

Esta línea se escribirá de derecha a izquierda
